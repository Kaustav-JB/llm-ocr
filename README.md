Vision OCR is a web-based Optical Character Recognition (OCR) application that extracts handwritten and printed Hindi and English text from images using Google's Gemini Vision models.

This project provides a modern frontend interface and a FastAPI backend integrated with the Gemini Vision API for accurate multilingual OCR.

---

## Features

- Extract handwritten and printed Hindi and English text
- Powered by Gemini Vision (gemini-3-flash-preview)
- FastAPI backend for high-performance processing
- Modern responsive frontend with drag-and-drop upload
- Real-time image preview
- Copy extracted text to clipboard
- Fully responsive layout
- Clean, professional UI
- Local backend execution with secure API key management

## Supported Input

- JPG
- JPEG
- PNG
- Handwritten text
- Printed text
- Hindi (Devanagari script)
- English
